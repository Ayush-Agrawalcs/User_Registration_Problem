# User Registration Validation System

## 📌 Introduction

The **User Registration Validation System** is a Python-based application that validates user input before account creation. It ensures that all user details meet predefined rules using **regular expressions (regex)**.

---

## 🚀 Features

* ✅ Validate user's **first name**
* ✅ Validate user's **last name**
* ✅ Validate user's **email address**
* ✅ Validate user's **phone number**
* ✅ Validate user's **password strength**

## 📖 Validation Rules (Example)

* **First/Last Name**: Starts with a capital letter, minimum 3 characters
* **Email**: Must follow standard email format (e.g., [abc@domain.com](mailto:abc@domain.com))
* **Phone Number**: Country code + number (e.g., +91 9876543210)
* **Password**:

  * Minimum 8 characters
  * At least one uppercase letter
  * At least one digit
  * At least one special character
